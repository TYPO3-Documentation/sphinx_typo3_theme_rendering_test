Compact ListsÂ¶

Keywords: css, compact-list

Add CSS class compact-list to lists to get li tags with margin-top:0; margin-bottom:0.

In case of a normal list:

.. rst-class:: compact-list

-  abc
-  bcd
-  cde

compact list

normal list

  • abc

  • bcd

  • cde

  • def

  • efg

  • abc

  • bcd

  • cde

  • def

  • efg

  1. one

  2. two

  3. three

  1. one

  2. two

  3. three

Or, for example:

.. rst-class:: compact-list

1. one
2. two
3. three

Should look like:

  1. one

  2. two

  3. three

In case of .. toctree:::

.. rst-class:: compact-list
.. troctree::

   Abc/Index
   Bcd/Index

In case of .. contents:::

.. contents::
   :class: compact-list
