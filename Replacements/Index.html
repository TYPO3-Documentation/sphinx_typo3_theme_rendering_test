About the settingsÂ¶

Note: Choose only one of the example lines for Settings.cfg.

|cfg_audience|Â¶

Will be replaced by the value of audience as defined in conf.py or the empty string.

Example definition in Settings.cfg:

[general]

audience = developers, power users
audience = Everybody interested
|cfg_author|Â¶

Will be replaced by the value of author as defined in conf.py. If empty, t3author will be used. If none is found, the empty string is used.

Example definition in Settings.cfg:

[general]

author = TYPO3 Documentation Team
author = Ernest Hemingway <ernest@example.org>, Oscar Wilde
